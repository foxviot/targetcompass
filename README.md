# targetcompass
TargetCompass V5 是一个本地运行的生信 Agent 证据链平台原型，用于把自然语言研究问题转化为可追溯的资源发现、方法路由、分析任务、QC、Evidence DB 与报告流程。系统包含 canonical agent protocol、EvidencePlan、DatasetProfile、MethodContract、Resource Gate、Artifact Registry、EvidenceRepository、Codex Worker 协议、Nextflow/本地执行适配和 PilotDeck Web UI，适合演示和继续开发自动化靶点发现工作流。
